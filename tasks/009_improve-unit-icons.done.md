# Improve Army and Fleet Unit Icons

## Goal
Add more visual detail to army (artillery) and fleet (battleship) icons while keeping the same bounding box.

## Resolution
- **Artillery**: Added wheels with axle bar, barrel reinforcement band, rivet detail on shield
- **Battleship**: Added portholes, forward gun turret, tapered bridge, crow's nest, tapered smokestack with smoke band detail
